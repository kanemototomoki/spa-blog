# リバースプロキシとは？

## クライアントとWEBサーバの間に入るサーバのこと
- セキュリティ向上のため(クライアントはプロキシサーバの存在しか認識できない)
- ロードバランサ機能も内包できる
- キャッシュ機能も内包できる

## 関わりのあるワード
- プロキシサーバ
- キャッシュサーバ
- ロードバランサ
- Nginx
- Apache
